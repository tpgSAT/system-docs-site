# レンタルサーバー

TPG公式ホームページ、WordPress等をホストしています。また、メールサーバーとしても利用しています。

## 基本情報

- ストレージ: 300GB
- 契約: さくらのレンタルサーバ スタンダード
- ログインページ: [https://secure.sakura.ad.jp/rs/cp/](https://secure.sakura.ad.jp/rs/cp/)
