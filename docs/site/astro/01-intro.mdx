# はじめに

## Astroってなに？

Astro（アストロ）は、モダンなフロントエンド開発に特化した静的サイト生成（SSG）を中心とするWebフレームワークです。
ReactやVueのような従来のSPAフレームワークと異なり、「**できる限りJavaScriptをクライアントに送らない**」ことを重視しています。
つまり、HTMLをサーバーサイドで生成し、クライアントには最小限のJavaScriptだけを送信することで、ページの読み込み速度を向上させ、SEO対策にも優れています。

## Astroの特徴

#### 1. デフォルトでJavascriptが0

AstroはHTMLを最初に生成し、クライアント側にJavaScriptをほとんど送らない（必要なものだけ）。
「静的HTML + 必要なところだけ動的に」が基本。

#### 2. Island Architecture（アイランドアーキテクチャ）

ページ全体をSPAとして扱うのではなく、一部のコンポーネントだけをインタラクティブにする。
例えば、ナビゲーションバーやチャットウィジェットなどだけをReact/Vueなどで動かせる。

#### 3. フレームワークに依存しない

React, Vue, Svelte, Preact, Solidなど、複数のUIフレームワークを混在して使える。

#### 4. マークダウンやMDXがネイティブ対応

ブログやドキュメントサイトの構築に最適。
Markdownにコンポーネントを埋め込むMDX形式もサポート。

#### 5. 高速なビルドと実行

Vite（高速ビルドツール）ベースの開発環境。
ビルドは非常に高速で、出力は最適化された静的ファイル。

## 書き方の例

```astro
---
// これはフロントマター
const title = "ようこそAstroへ！"
---

<html>
  <head>
    <title>{title}</title>
  </head>
  <body>
    <h1>{title}</h1>
    <p>静的なHTMLを生成します！</p>
  </body>
</html>
```
