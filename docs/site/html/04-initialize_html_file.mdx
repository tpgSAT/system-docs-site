# HTML の初めに書くこと

前のページで、HTML/CSS ファイルを作成しました。では、実際に書いていきます。

まず、HTML ファイルに以下のコードをコピペします。



```html
<!DOCTYPE html>
<html>
  <head>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="読み込みたいCSSファイルのパス" />
    <title>ページのタイトル</title>
  </head>
  <body></body>
</html>
```



　これはHTMLファイルを作成したら必ず書くことです。覚える必要はなく、毎回コピーするので十分です。

　行の初めに空白が入っているところがあります。基本的にこの空白をインデントと呼びます。基本的にインデントは半角スペース4つ単位ですが、誤ってインデントをなくした場合など自分でインデントを作成したいときはTabキーを押してください。基本的には改行すれば適切な位置に自動でインデントが入るのでそんなに気にすることもないです。



　「ページのタイトル」とある部分は、ブラウザでページを表示したときにタブに入るタイトルの部分です。今回は「Taki Fes 2023 | Taki Plaza Gardener」と書き換えます。

　その次の行は、画面表示に関わる行なのですが基本設定のようなものなので書き換える必要はないです。

　さらにその次の、meta charsetと書いてある行は、使用する文字コードを指定しています。ここも書き換える必要はありません。



　「読み込みたいCSSファイルのパス」とある部分には、「./takifes.css」と書いてください。HTMLファイルにはページの内容そのものを書き、CSSファイルはページの見た目を整えるために使います。そのためにHTMLファイルとCSSファイルを紐付けておくための行です。パスはファイルの位置を示すものだと思ってもらえれば大丈夫です。パスの書き方やより詳しい説明については<a href="#">こちら</a>を参照してください。


　「\< \>」で挟まれたものをタグといいます。`<html></html>`はhtmlタグ、`<head></head>`はheadタグという風に今後書いていきます。headタグやhtml、bodyタグには、「/」がついているものとついていないものがあります。/ がついていないものを開始タグ、ついているものを閉じタグとよび、開始タグと閉じタグの間を「タグの中」と今後書いていくことにします。開始タグがあるのに閉じタグがないと正しく表示されないことがあるので気をつけてください。何かうまくいかないと思ったら1割くらいの原因はこのせいです。

head タグの中にある 4 行は、ページの情報を表しています。場合に応じて head タグの中は増えていきます。

ページに表示する文字などを書いていくのは、body タグの中です。body タグの間にカーソルを置き、改行してみます。自動でインデントが生成されるはずです。

　前回のページで、ホームページの画面構成はヘッダー、メイン、フッターだと書きました。ヘッダーとフッターには、専用のタグがあります。body タグの中に`<header></header>`と書いてください。このとき、Auto Close Tag というプラグインを入れてあるため、閉じタグは自動で生成されるはずです。(\< \>を先に書いて中に header と書くのではなく、左から順に \< header と書いて、その後 \> を入力するとうまくいくと思います。)
　改行して、footer タグも書いてください。メインに対応するタグはなく、メインは header タグと footer タグの間に書いていきます。

今回は、このようになっていれば大丈夫です。

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Taki Fes 2023 | Taki Plaza Gardener</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta charset="UTF-8" />
    <link rel="stylesheet" href="./takifes.css" />
  </head>
  <body>
    <header></header>

    <!-- ここにメイン部分を書く -->

    <footer></footer>
  </body>
</html>
```

補足ですが、薄い文字になっている<!-- -->部分はコメントアウトといって、ページに影響を与えない部分です。なにを書いてあるのかなどをメモ書きするときにつかいます。コメントアウトをしたい行にカーソルを置いて Ctrl + / を押すとコメントアウトできます。
