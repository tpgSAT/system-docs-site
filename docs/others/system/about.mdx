# 概要

## システム班はどういう班？

システム班はWebサイトやTPG内部向けITインフラを保守・管理するチームです。

具体的には、[TPG 公式サイト](https://tpgd.jp/)の作成、[若葉祭公式サイト](https://wakabasai.tpgd.jp/)の作成、TPG Portal の保守・管理をしています。
そのほかに、システム班で得た知見や仕事マニュアルを保管しておくためのサイト（このサイトのことです）の管理などもしています。

また、システム班は2024年度に発足した比較的新しい班で、それ以前はSAT(System Administrator Team)という名前でマネジメント班内で活動していました。

## システム班の活動イメージ

システム班のメンバーは、TPGに所属する3年間（正確には2年半）で以下のような流れで活動を進めていきます。  
ゼロからスタートの人を想定したものを一応作ってみましたが、順番やペースは履修とか兼サーとかも考慮して自由な感じで  
Web 系とサーバー系をどっちもできる人になるルートなので、所属メンバーが十分に居れば分担してしまうのも全然あり


- 最初はこんな感じでスタート

  - このサイトの「はじめに」を読んで SAT について把握する ← 今ここ
  - GitHub を使って Web サイトを作る練習をする（GitHub の使い方を覚える）
    - [勉強用資料/Web 系](/learn/website/README.md)
  - 実際に Web ページの更新作業に携わる
  - TPG 公式サイト「Whats New」と「Blog」の更新ができるようになる
  - （自信がない人はタイピングの練習もやってみるといいかも。[e-typing](https://www.e-typing.ne.jp/)でランクが「C-」以上なら SAT ないし大学では十分だと思います。）
  - 気になる人は「IT パスポート」という資格の勉強も始めてみるといいかも

- 速い人だと 2Q くらいからこんなことに手を出し始める

  - TPG Portal が使えるようになる（ほかのメンバーに教えられる）
  - TPG Portal の管理者モード(?)をマスターし始める（季節に合わせた見た目にできるのもココ）

- 3Q 以降に・・・

  - Web ページの更新作業を担当できるようになる（難しいことをしなければゼロからページを作れるようになる）
  - レンタルサーバーに手を出し始める
  - レンタルサーバーの保守・管理ができるようになる（レンタルサーバーと GitHub のつながり方とかもわかるようになる）

- もうすぐ 2 年生になるって頃には・・・

  - VPS に手を出し始める

- 2 年生

  - VPS の保守・管理ができるようになる（3 年生が居る 3Q 途中までに行きたい）
  - エラーとかを解決したりする
  - 蓄積した知識とかをこのサイトに書いて後の学年に残す
  - 1 年生にいろいろアドバイスしたりとか

- 3 年生
  - 1 ～ 2 年生のバックアップ
  - エラーが起きた時などに 2 年生と一緒に緊急対応したりする
  - 蓄積した知識や経験をこのサイトや後の学年に伝えていく
  - TPG の卒業が秋なので、秋学期中に 1・2 年生が困らないようにする
